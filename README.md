# :point_right:SpringBoot, JPA, mustache를 이용한 BookStore:point_left:

개인 프로젝트로 JAVA를 이용하여 온라인 서점을 구축.

데이터베이스는 AWS의 RDS를 사용하여 구현.

## 구현 기술:computer:
* Spring boot
* JPA
* Javascript
* MySQL
* AWS RDS
* mustache
* build.gradle
* Spring security(암호화)
